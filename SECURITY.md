# Security Policy

## Educational Reference Material

This repository contains the pedagogical text and sample snippets for the **Python Under the Hood** handbook.
Because this project is purely an educational resource and does not distribute a compiled, production-ready software package or executable library, there are no live versions to patch against software vulnerabilities.

## Reporting Technical Errors

If you notice an error in a CPython architectural explanation, a broken code snippet, or a typo that could confuse students, please do not treat it as a security exploit.
Instead, feel free to open a regular **Public Issue** or submit a **Pull Request** directly in this repository so we can improve the book for the entire community!
